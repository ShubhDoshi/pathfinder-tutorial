How To Use Pathfinding Visualizer ?

  • The position of the Start Node and the Target Node is fixed.

  • Click on any node of the grid to create wall at that node. Drag and hold to create multiple walls.

  • You can form various mazes and patterns from the options available in the Mazes and Patterns menu which are formed by using various algorithms.

    The following options are available to form mazes and patterns:

  • Maze using Kruskal's Algorithm
  • Maze using Depth First Search
  • Maze using Prim's Algorithm
  • Pattern formed by adding vertical walls
  • A random pattern or maze

    • (Note: The Random Maze Generator (last option) doesn't always guarantee a path from Start Node to Target Node as the path might be blocked by the walls.)

  • You can find the shortest path from start node to target node through various algorithm options available in the Algorithms menu.

    The following options are available to find shortest path from start node to target node:

  • Dijkstra Algorithm
  • Breadth First Search
  • A*(A-star) Search Algorithm
  • Greedy Search Algorithm
  • Depth First Search

    • (Note: Greedy Search Algorithm and Depth First Search don't always give the shortest path from start node to target node.)

  • Click on Visualize button to find the shortest path from the Start Node to the Target Node using Dijkstra's Algorithm.

  • To refresh the page, click on the Pathfinding Visualizer button at the top-left corner.

  • To clear the grid, click on Clear Board button.

  • Select the speed of visualization from the Speed menu to visualize the working of Dijkstra's Algorithm with different speeds.
